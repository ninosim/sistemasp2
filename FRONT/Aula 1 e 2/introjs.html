Comandos JavaScript

Um programa JavaScript é uma lista de comandos a serem executados por um computador.

Comandos JavaScript são compostos de:

  • Valores
  • Operadores
  • Expressões
  • Palavras-Chave
  • Comentários

Valores

A sintaxe JavaScript define dois tipos de valores:
  • Literais
  • Variáveis
